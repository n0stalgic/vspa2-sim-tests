Detailed Description

High-speed AXIQ driver.

Author
NXP Semiconductors.

This file provides a software interface for the High-Speed AXI I/Q interface AXIQ_HS. The hardware includes a single AXIQ_HS module that is, from a software perspective, perceived as 2 banks (AXIQ_HS_BANK_0 and AXIQ_HS_BANK_1). Each bank provides 1 receive channel (AXIQ_HS_FIFO_RX) and 1 transmit channel (AXIQ_HS_FIFO_TX).
