Detailed Description

Description:
This chapter documents scalar arithmetic unit intrinsics. Only few intrinsics are supported for scalar instructions and these are for instructions that cannot be expressed easily in C.

Prototype:
void __ff1(Type1 bit_num_set, Type2 val)
Description:
Find first one
Move into bit_num_set the bit number of the most significant bit in val
that is set.
Type1 can be int, or unsigned int
Type2 can int, unsigned int, or float
Underlying assembly:
ff1 GP,GP1

VSPA

Prototype:
void __log2(Type log_result, float val)
Description:
Compute base-2 logarithm
Compute base-2 logarithm of a single precision floating point number
in val and store in log_result.
The format of the fixed-point result will be in Q16.15 format i.e. 1 sign bit,
16 integer bits and 15 fractional bits.
Underlying assembly:
log2 GP,GP1
Support:
VSPA
