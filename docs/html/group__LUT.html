Prototype:
void __lut_full_wr(Type *__addr1, const short __addr2)
Description:
Copy a full table from LUT RAM to LUT pages
Copy a full table from LUT RAM at address addr1 to LUT pages, then post-increment
the address by addr2. addr1 is post-incremented by half-table length. addr1 should
be aligned to a full-table. Type can be a array address of 256 elements; an element
being a complex type or two data values.
Underlying assembly:
lut.fwr [gX]+Is6
Support:
VSPA2

Prototype:
void __lut_invert_htp()
Description:
Invert LUT half table page pointer bit
Switch the Look Up table read page pointer bit for half table. This controls
which half of the buffer is written by the half table write instruction.
Underlying assembly:
lut.hsw
Support:
VSPA2
