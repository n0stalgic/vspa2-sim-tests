Prototype:
void __fa0to1(const unsigned int H_QUADRANT)
Description:
Find all zero to one
Returns single bit for each adjacent pair of 0-1 bits in a quadrant of
H register. H_QUADRANT must be an immediate constant between 0 and 3. 
The result is stored in the dH register which can be accessed with the
bin2num instruction.
Underlying assembly:

Prototype:
void __ff0to1(Type val1, Type val2)
Description:
Find first zero to one
Move into val1 the bit position of the first 0-to-1 transition in the
H register, then skip ahead val2 bit positions. Let's call the value
at this bit position skip_bit_val. All the bits in H up to this point
are set to skip_bit_val.
Type can be an int or an unsigned int.
Underlying assembly:
ff0to1 GP,GP1
