Prototype:
void *__modulo_ptr_add(void *ptr, ptrdiff_t incr, void *base, size_t mod)
Description:
Use modulo addressing.
Use modulo addressing to manage circular buffers. The intrinsic computes and
returns the updated value of ptr based on the start of the buffer (base),
the increment value (incr) and the size for the wrap around (mod), as follows:

ptr = base + (((ptr - base) + incr) % mod)
 
The increment and the modulo is multiplied by the size of the pointer type, 
which is 2 for int, 1 for char & short on vspa2
The modulo and increment are expressed in number of elements, no matter the size.

N/A
Support:
VSPA

Initialize the bit reversal state machine
Initialize the bit reversal state machine with a specific address and fft size.
The address will be stored in an internal bit reversal address register
and will be used by the next store or load with bit reversal.
Type can be any type in the Scalar data types section defined in 2.1.
