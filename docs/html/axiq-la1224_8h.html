Detailed Description

LA1224 AXI I/Q FIFO driver interface.

Author
NXP Semiconductors.

This file provides a software interface for the 3 AXI I/Q hardware interfaces in LA1224. The 2 low-speed AXIQ interfaces include 2 receive paths and 2 transmit paths. The high-speed AXIQ interface is perceived as a 2-bank module with 1 receive path and 1 transmit path per bank.
