Prototype:
void __set_xtrm(UNS1b s, MIN1b m, EV1b cmp, IV iv, const unsigned int n)
Description:
Configure the extrema search engine
Configure the extrema search engine. This includes the number of half-word
elements to search, n, which can be any power of 2 up to one half-line
(__AU_COUNT__ * 2) or a multiple of half-line up  to 64 * (__AU_COUNT__ * 2).
The search engine uses rS2[msb:msb-3] to read vector data from the VRA.
In 'all' mode, n must be greater than or equal to 2 and in 'even' mode,
n must be greater than or equal to 4.
UNS1b: { signed, unsigned }
MIN1b: { max, min }
EV1b:  { all, even }
IV:    { index, value }
Underlying assembly:
set.xtrm UNS1b,MIN1b,EV1b,IV,IM12
Support:
VSPA2

Prototype:
unsigned int __xtrm_custom(__index, __value)
Description:
Return the index and value of the extrema
Start an extrema search and place the index and value of the extrema into a
scalar register. The search on the list of elements is done on 16-bit fixed
data types. This is a multi-cycle instruction that depends on the number of
elements in the search buffer.
Underlying assembly:
xtrm GP
Support:
VSPA
