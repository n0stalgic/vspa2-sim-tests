◆ __assert_au_precision

#define __assert_au_precision (   __a)    __asm (".setprec " #__a);
Prototype:
void __set_au_precision(Precision prec)
Description:
Assert arithmetic unit precision
This intrinsic does not generate any instruction. It asserts the
arithmetic unit precision mode to help the backend scheduling component
of the compiler to generate efficient and correct code. The precision
can be set to single or double. The user should use this assert when
the precision is different from the expected one.
Note that the arithmetic unit default precision is single. Hence, any
use of double without explicitly calling the intrinsic
set_prec(..., ..., ..., double, ...) should be preceded by
assert_au_precision(double) and this includes functions that are
expected to be called with the double mode.
Precision: {single, double}

Example:
void foo()
{
   ... // code running in single precision mode

   test();
   // test is a function that intentionally sets the au precision mode to
   // double and does not restore it back to single before returning.

   __assert_au_precision(double);

   ...  // code expected to run in double precision mode.

}
Underlying assembly:
compiler directive
Support:
VSPA

Description:
Set precision for the vector data path
Set precision for the vector data path. This includes source registers
(S0, S1, S2), AU operations and and AU write-results.
S_PREC:  {half_fixed, half, single, double }
AU_PREC: { single, double }
WB_PREC: {half_fixed, half, single, double }

◆ __set_prec_single

#define __set_prec_single ( )    __set_prec(single, single, single, single, single);
Prototype:
void __set_prec_single()
Description:
Set single precision for all vector data path
Set single precision for all vector data path. This includes source
registers (S0, S1, S2), AU operations (v), and AU write (St).
This is a common use of the general intrinsic __set_prec(). It is
equivalent to __set_prec(single, single, single, single, single)
Underlying assembly:
set.prec single,single,single,single,single
Support:
VSPA
