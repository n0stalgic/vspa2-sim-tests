Prototype:
unsigned int __ip_read(const unsigned int ipreg, const unsigned int mask) unsigned int __ip_read(const unsigned int ipreg)
Description:
Move from an IP register to scalar
__ip_read(ipreg, mask): Read from the IP register ipreg using the mask
value. The mask determines the parts of the IP register to be read.

__ip_read(ipreg): Read from the IP register ipreg using the mask
value of 0xFFFFFFFF and hence it reads the complete IP register.
This is a special version of the intrinsic __ip_read(__ipreg, mask).
Underlying assembly:
mvip GP,IP_REG,IM32 mvip GP,IP_REG,GP1
Support:
VSPA

Prototype:
void __ip_write(const unsigned int ipreg, unsigned int mask, unsigned int val) void __ip_write(const unsigned int ipreg, unsigned int val)
Description:
Write a scalar to an IP register
Write the value val to the IP register ipreg using the mask value mask.
If the mask value is 0xFFFFFFFF, then the complete IP register is
modified. The instruction generated for this case is either
mvip IP_REG, IM32 or mvip IP_REG, GP, 0xFFFFFFFF and that depends on
whether val is a constant or a variable. For all other mask values,
a part of the IP register is modified and the instruction generated
is mvip IP_REG, GP, IM32
Underlying assembly:
mvip IP_REG,GP,IM32 mvip IP_REG,GP,GP1
Support:
VSPA
